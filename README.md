# sports_image-classification
Objective is to identify sports in each category for classification. 
Hundreds of sports exist, for some games players should participate in single contestants, through for some with hundreds of simultaneous participants in teams playing as individuals. As there are so many types of sports worldwide, it’s quite difficult for people to know the names for all types of sports. Problems will arise when there are those who want to venture into the field of sports because it looks very attractive after watching it on television but do not know the name or category of the sport. Also, because there are several sports that look similar but are different, make it quite difficult for people to differentiate them. As a computer science student,I may try to help, to solve this problem by creating a sports classification system. This system helps people to identify the criteria of some unknown sports so that all people can know the names of sports they like.
